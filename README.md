# 台湾综合所得税计算器 2025

> 基于2025年最新税制规则的台湾综合所得税计算器，提供详细的税务计算、节税建议和完整的税额计算公式说明。

## 🎯 功能特色

- ✅ **2025年最新税制** - 完全依据财政部公布的2025年税务规则
- ✅ **详细计算过程** - 显示完整的税额计算公式和步骤
- ✅ **智能扣除额计算** - 自动计算各项免税额、扣除额和特别扣除额
- ✅ **完整扣除额支持** - 支持所有可选择的扣除额项目
- ✅ **已婚分开计税** - 支持三种计税方式，自动选择最省税方案
- ✅ **节税建议** - 提供合法的节税策略和建议
- ✅ **即时计算** - 输入数据即时显示计算结果
- ✅ **数据缓存** - 自动保存输入数据，页面刷新不丢失
- ✅ **响应式设计** - 支持桌面和移动设备

## 🔥 2025年税制重点变化

### 幼儿学前特别扣除额大幅调整
- 适用年龄从5岁以下**扩大至6岁以下**
- 第2名子女起享有**加成50%**（22.5万元）
- **取消排富规定**，所有家庭均可适用

### 房屋租金改列特别扣除额
- 从列举扣除额改为**特别扣除额**，申报更简便
- 每户每年**最高18万元**
- 适用租屋自住且**无自有住宅者**

### 各项金额全面调升
- **免税额**：每人9.7万元（+5,000元）
- **标准扣除额**：单身13.1万元（+7,000元）、夫妻26.2万元（+14,000元）
- **薪资及身心障碍扣除额**：各21.8万元（+11,000元）
- **基本生活费**：每人21万元（+8,000元）

## 💰 计算器功能说明

### 基本信息输入
- 薪资收入（自动扣除薪资特别扣除额）
- 其他收入（利息、股利、租金等）
- 婚姻状况（单身/已婚合并申报）
- 已婚计税方式（合并/薪资分开/全部分开/自动选择）

### 扶养亲属
- **6岁以下子女**（享有幼儿学前特别扣除额）
- **一般扶养亲属**（6岁以上70岁以下）
- **70岁以上长辈**（享有较高免税额）
- **大专院校学生**（教育学费特别扣除额）
- **身心障碍人士**（身心障碍特别扣除额）
- **长期照顾需求者**（长期照顾特别扣除额）

### 特别扣除额
- **房屋租金支出**（2025年新制特别扣除额，每户18万元）
- **储蓄投资利息**（每户27万元）
- **长期照顾特别扣除额**（每人12万元，有排富规定）

### 列举扣除额（与标准扣除额择一适用）
1. **捐赠**
   - 一般捐赠：限所得总额20%
   - 政府、国防、文化资产捐赠：无金额限制

2. **人身保险费**
   - **非健保保费**：每人限24,000元（人寿、伤害、年金保险等）
   - **全民健保费**：无金额限制，可全额扣除

3. **医疗及生育费用**
   - 无金额限制，核实认列
   - 限公立医院、健保特约医院

4. **灾害损失**
   - 不可抗力灾害造成的财产损失
   - 无金额限制，需扣除保险理赔

5. **自用住宅购屋借款利息**
   - 每户限30万元
   - **重要**：需先扣除储蓄投资特别扣除额

### 计算结果显示
- **应纳税额**及有效税率
- **税后净收入**
- **适用税率级距**
- **完整计算步骤**（薪资净额→扣除额→综合所得净额→应纳税额）
- **扣除额明细分项**
- **已婚分开计税比较**（自动选择最省税方案）

## 🎯 节税建议

1. **扣除额规划** - 善用各项特别扣除额，特别是2025年大幅调升的幼儿学前扣除额
2. **保险规划** - 人身保险费每人每年最高2.4万元，健保费无限制全额扣除
3. **捐赠节税** - 合法捐赠可列举扣除，一般限所得总额20%
4. **投资理财** - 储蓄投资特别扣除额每户27万元
5. **房屋租金** - 租屋族新福利，房租改列特别扣除额，每户18万元
6. **长期照顾** - 符合条件者每人可扣除12万元（有排富规定）
7. **已婚计税** - 系统自动比较三种计税方式，选择最省税方案

## 💡 重要提醒

### 购屋借款利息计算规则
购屋借款利息扣除额 = 支付利息 - 储蓄投资特别扣除额（最高30万元）

**例如**：
- 房贷利息：30万元
- 储蓄利息：2.8万元
- 实际可扣除房贷利息：30万 - 2.8万 = 27.2万元

### 已婚计税方式说明
1. **合并计税**：夫妻所得合并，适用同一税率级距
2. **薪资分开计税**：本人薪资单独计税，其他所得合并计税
3. **全部分开计税**：夫妻各自计税，各适用不同税率级距
4. **自动选择**：系统计算三种方式，自动选择税负最低者
