# 台灣綜合所得稅計算器 2025

> 基於2025年最新稅制規則的台灣綜合所得稅計算器，提供詳細的稅務計算、節稅建議和完整的稅額計算公式說明。

## 🎯 功能特色

- ✅ **2025年最新稅制** - 完全依據財政部公布的2025年稅務規則
- ✅ **詳細計算過程** - 顯示完整的稅額計算公式和步驟
- ✅ **智慧扣除額計算** - 自動計算各項免稅額、扣除額和特別扣除額
- ✅ **完整扣除額支援** - 支援所有可選擇的扣除額項目
- ✅ **已婚分開計稅** - 支援三種計稅方式，自動選擇最省稅方案
- ✅ **節稅建議** - 提供合法的節稅策略和建議
- ✅ **即時計算** - 輸入資料即時顯示計算結果
- ✅ **資料快取** - 自動保存輸入資料，頁面重新整理不遺失
- ✅ **響應式設計** - 支援桌面和行動裝置

## 🔥 2025年稅制重點變化

### 幼兒學前特別扣除額大幅調整
- 適用年齡從5歲以下**擴大至6歲以下**
- 第2名子女起享有**加成50%**（22.5萬元）
- **取消排富規定**，所有家庭均可適用

### 房屋租金改列特別扣除額
- 從列舉扣除額改為**特別扣除額**，申報更簡便
- 每戶每年**最高18萬元**
- 適用租屋自住且**無自有住宅者**

### 各項金額全面調升
- **免稅額**：每人9.7萬元（+5,000元）
- **標準扣除額**：單身13.1萬元（+7,000元）、夫妻26.2萬元（+14,000元）
- **薪資及身心障礙扣除額**：各21.8萬元（+11,000元）
- **基本生活費**：每人21萬元（+8,000元）

## 💰 計算器功能說明

### 基本資訊輸入
- 薪資收入（自動扣除薪資特別扣除額）
- 其他收入（利息、股利、租金等）
- 婚姻狀況（單身/已婚合併申報）
- 已婚計稅方式（合併/薪資分開/全部分開/自動選擇）

### 扶養親屬
- **6歲以下子女**（享有幼兒學前特別扣除額）
- **一般扶養親屬**（6歲以上70歲以下）
- **70歲以上長輩**（享有較高免稅額）
- **大專院校學生**（教育學費特別扣除額）
- **身心障礙人士**（身心障礙特別扣除額）
- **長期照顧需求者**（長期照顧特別扣除額）

### 特別扣除額
- **房屋租金支出**（2025年新制特別扣除額，每戶18萬元）
- **儲蓄投資利息**（每戶27萬元）
- **長期照顧特別扣除額**（每人12萬元，有排富規定）

### 列舉扣除額（與標準扣除額擇一適用）
1. **捐贈**
   - 一般捐贈：限所得總額20%
   - 政府、國防、文化資產捐贈：無金額限制

2. **人身保險費**
   - **非健保保費**：每人限24,000元（人壽、傷害、年金保險等）
   - **全民健保費**：無金額限制，可全額扣除

3. **醫療及生育費用**
   - 無金額限制，核實認列
   - 限公立醫院、健保特約醫院

4. **災害損失**
   - 不可抗力災害造成的財產損失
   - 無金額限制，需扣除保險理賠

5. **自用住宅購屋借款利息**
   - 每戶限30萬元
   - **重要**：需先扣除儲蓄投資特別扣除額

### 計算結果顯示
- **應納稅額**及有效稅率
- **稅後淨收入**
- **適用稅率級距**
- **完整計算步驟**（薪資淨額→扣除額→綜合所得淨額→應納稅額）
- **扣除額明細分項**
- **已婚分開計稅比較**（自動選擇最省稅方案）

## 🎯 節稅建議

1. **扣除額規劃** - 善用各項特別扣除額，特別是2025年大幅調升的幼兒學前扣除額
2. **保險規劃** - 人身保險費每人每年最高2.4萬元，健保費無限制全額扣除
3. **捐贈節稅** - 合法捐贈可列舉扣除，一般限所得總額20%
4. **投資理財** - 儲蓄投資特別扣除額每戶27萬元
5. **房屋租金** - 租屋族新福利，房租改列特別扣除額，每戶18萬元
6. **長期照顧** - 符合條件者每人可扣除12萬元（有排富規定）
7. **已婚計稅** - 系統自動比較三種計稅方式，選擇最省稅方案

## 💡 重要提醒

### 購屋借款利息計算規則
購屋借款利息扣除額 = 支付利息 - 儲蓄投資特別扣除額（最高30萬元）

**例如**：
- 房貸利息：30萬元
- 儲蓄利息：2.8萬元
- 實際可扣除房貸利息：30萬 - 2.8萬 = 27.2萬元

### 已婚計稅方式說明
1. **合併計稅**：夫妻所得合併，適用同一稅率級距
2. **薪資分開計稅**：本人薪資單獨計稅，其他所得合併計稅
3. **全部分開計稅**：夫妻各自計稅，各適用不同稅率級距
4. **自動選擇**：系統計算三種方式，自動選擇稅負最低者
